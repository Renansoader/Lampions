# Lampions Marketing

Site institucional desenvolvido para a **Lampions Marketing**, agência de marketing digital sediada em Recife/PE.

## Funcionalidades

- Landing page com apresentação da agência, serviços e cases de sucesso
- Calculadora de ROI interativa para apresentação a potenciais clientes
- Design responsivo adaptado para mobile e desktop
- Integração com WhatsApp para contato direto

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Landing page principal |
| `roi-calculator.html` | Calculadora de ROI interativa |
| `assets/style.css` | Estilos compartilhados |
